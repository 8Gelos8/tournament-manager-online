# Tournament Manager Online

Онлайн система управління турнірами з підтримкою множини користувачів.

## Технології
- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Google Generative AI

## Розгортання
1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
3. Налаштуйте змінні середовища в `.env.local`
4. Запустіть: `npm run dev`
